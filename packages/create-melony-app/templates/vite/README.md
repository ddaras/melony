# Melony Vite App

This is a [Vite](https://vitejs.dev) + React project bootstrapped with [create-melony-app](https://github.com/ddaras/melony).

## Getting Started

First, install dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.

**Note:** Make sure you have a backend server running on `http://localhost:3000/api/chat` or update the URL in `src/App.tsx`.
