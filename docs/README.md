# Melony Documentation

Welcome to the Melony documentation. Melony is a universal AI agent framework with built-in Server-Driven UI (SDUI) and Human-in-the-Loop (HITL) support.

## Table of Contents

- [**Introduction**](./introduction.md): Overview of what Melony is and why it exists.
- [**Getting Started**](./getting-started.md): Installation and your first "Hello World" agent.
- [**Core Concepts**](./core-concepts.md): Deep dive into Events, Actions, Hooks, and Plugins.
- [**Server-Driven UI (SDUI)**](./server-driven-ui.md): Building rich interactive interfaces from the server.
- [**Adapters & Client**](./adapters.md): Serving your agent and consuming it on the frontend.
- [**Advanced Usage**](./advanced-usage.md): HITL, Persistence, and custom UI components.

## Recommended Tech Stack

To get the most out of Melony, we recommend the following stack:

- **Backend**: [Hono](https://hono.dev) (runs anywhere: Cloudflare Workers, Node, Deno).
- **Frontend**: [React](https://reactjs.org) with `@melony/react`.
- **Validation**: [Zod](https://zod.dev) for typed action parameters.
